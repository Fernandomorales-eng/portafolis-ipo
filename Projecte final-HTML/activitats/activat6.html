Llista de definició

Objectiu: Fer una llista amb termes relacionats amb el món web i la seva definició.

HTML
Llenguatge de marques utilitzat per a estructurar i organitzar el contingut d'una pàgina web.
CSS
Llenguatge de fulls d'estil utilitzat per a definir l'aspecte visual, disseny i maquetació d'un document HTML.
JavaScript
Llenguatge de programació que permet afegir interactivitat, dinamisme i funcions complexes a les pàgines web.
Navegador
Programari o aplicació (com Chrome o Firefox) que interpreta el codi HTML, CSS i JS per mostrar les pàgines web a l'usuari.
Pàgina web
Document electrònic adaptat per a la World Wide Web que conté informació i al qual s'accedeix mitjançant un navegador.
E-commerce
Sistema de compra i venda de productes o serveis a través d'Internet i plataformes digitals.
W3C
World Wide Web Consortium, l'organisme internacional encarregat de desenvolupar els estàndards i recomanacions per a la web.
Internet
Xarxa global de lComputadors interconnectats que permet l'intercanvi d'informació i serveis a escala mundial.
